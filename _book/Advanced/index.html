前端进阶

闭包

如果在函数里面可以访问外面的变量，那么这个函数+这些变量 = 闭包

闭包+时间

请问下面打印除什么？

for(var i=0;i<6;i++) {
    setTimeout(
        () => console.log(i)
    )
}

6个6。因为六个函数公用一个 i

如果需要打印0，1，2，3，4，5

for(let i= 0;i<6;i++) {
    setTimeout(
        () => console.log(i)
    )
}
