闭包

闭包

闭包是什么？

闭包是指有权访问里一个函数作用域链中的变量的函数。-- JavaScript高级程序设计（红皮书）

也就是如果在函数里面可以访问外面的变量，那么这个函数+这些变量 = 闭包。

看看例子：

function A(){
  var a = 1
  function B(){
    console.log(a)
  }
  return B
}

var fn = A()
fn() //1

fn()就是调用函数A()，函数A()执行了函数B。函数B执行的时候通过作用域链找到变量a。那么函数B和它能访问到的变量a之和就是闭包。

需要注意的是。闭包所保存的是整个变量对象。也就是

{a : 1}

如果a执行过程中发生改变，闭包中的a也改变。

function A(){
  var a = 1
  function B(){
    console.log(a)
  }
  a = 3   // 修改 a 值
  return B
}

var fn = A()
fn() //3

总结

那闭包的特点呢？

  • 能函数维持住变量
  • 但是不能维持变量的值
